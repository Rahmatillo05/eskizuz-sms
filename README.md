PHP Eskiz.uz
============

PHP Eskiz.uz SMS Gateway simple package

## Installation

```bash
composer require rahmatullo/eskiz-sms "^1.5"
```

or add the following to your `composer.json` file:

```json
{
    "require": {
        "rahmatullo/eskiz-sms": "^1.5"
    }
}
```
